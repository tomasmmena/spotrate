Welcome to Spotrate!

This small sample application implements a simple API that takes JSON and XML and returns the rate for a parking spot given a start and end datetimes.

Endpoints

POST /api/json

This endpoint processes requests submitted as JSON.

Example

Request

{
	"start": "2018-05-06 11:30:52",
	"end": "2018-05-06 15:00:03"
}
Response

{
    "rate": 2000
}

POST /api/xml

This endpoint processes requests submitted as XML.

Example

Request

<request>
	<start>2018-05-06 11:30:52</start>
	<end>2018-05-06 15:00:03</end>
</request>
Response

<response>
  <rate>2000.0</rate>
</response>
}
